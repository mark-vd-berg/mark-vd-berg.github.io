Over

Ik heb dit project gemaakt voor Informatica
Voor de informatie hebben we gebruik gemaakt van de API van buieradar en voor de navigatie.
Voor de oranje floating action button, de navigatie, de kleuren, de checkbox en de schuifjes heb ik materialize gebruikt.
Naast materialize heb ik hulp gehad van mijn vader, die heeft alles op github gezet en is de eigenaar van de site waarop mijn code word gehost.
Mijn site is altijd bereikbaar.



Let op

Na feedback van verschillende gebruikers, ben zijn wij erachter gekomen dat onze op wellicht niet werkt.
Tot nu toe hebben alleen Apple users dit probleem gehad maar het kan ook bij andere devices gebeuren.
Als de site bij u niet werkt, controleer dan of uw browser toegang heeft tot uw locatie, Apple geeft de browser standaard geen toegang,
hierdoor kan de website geen weersinformatie ontvangen.

Hartelijk dank, team Weerrader
